# AC'SCENT WOW (FragranceCaster)

AI가 당신의 최애를 분석하고 어울리는 향수를 추천하는 웹 애플리케이션입니다.

## 기능

- 최애 이미지 업로드 및 분석
- AI 기반 밈스러운 분석 결과 제공
- 최애와 어울리는 향수 추천
- 다국어 지원 (한국어, 영어, 중국어, 일본어)

## 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
- i18n

## 로컬 개발 환경 설정

1. 저장소 클론
```bash
git clone [저장소 URL]
cd acscent-universe-reboot/apps/wow3
```

2. 의존성 설치
```bash
pnpm install
```

3. 개발 서버 실행
```bash
pnpm dev
```

4. 브라우저에서 확인
```
http://localhost:3000
```

## 빌드 및 배포

프로덕션 빌드를 생성하려면:
```bash
pnpm build
```

빌드된 앱을 로컬에서 실행하려면:
```bash
pnpm start
```
